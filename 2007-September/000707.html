> How can I tell ion the kpdf windows is a transient windows ? 

It *is* a transient window, but it shouldn't be.  You can override that
by adding this to cfg_ion.lua:

defwinprop {
   class = "Kpdf",
   role = "presentationWidget",
   transient_mode = "off"
}
