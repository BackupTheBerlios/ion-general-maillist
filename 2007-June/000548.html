I believe he means that you would need to modify Firefox to apply
different window properties to each window.  This is somewhat
complicated because the C++ code that creates the dialogs may likely not
know anything about what its purpose will be, while the JavaScript code
(the UI of Mozilla browsers is written in JavaScript, essentially) that
actually knows the purpose of the window doesn't know anything about
X11, and thus isn't prepared to set specific window properties.

One hack may be to match the title of the window somehow.
