I'm using an old version of ion3 (ion-3ds-20060317)
which I compiled from source on Mac OS 10.4 (PPC)
and it works just fine. Haven't tried any newer versions.

To run it satisfactorily, you really need a full screen
X session. I have used both Apple's xquartz and
the xorg that is currently in Fink. I use xorg now.
You will probably need to fiddle with your xmodmap and
your key bindings to get comfortable keyboard
control. I could send you mine but the chances are
the key codes will be different on your computer anyway.
I like to use the 'command' keys as mod1 under ion,
and you can use xev to find out what key codes are
generated by each key.

Running more than one X session appears to be possible
(I've done it by accident) but is probably not a good idea.

Samer.
